DBFlute と Eclipse

DBFluteは、利用上において特に Eclipse に依存してはいません。ですが、Eclipse 上で DBFlute の利用を支援する EMecha が提供されているなど、Eclipseでの開発を支援する仕組みがあります。

Eclipseの操作

隠しファイルの設定

Package Explorer上に表示する/しないファイルを調整できます。

例えば、. (ドット)で始まるファイルやディレクトリはデフォルトでは表示されないので、よく意識するようであれば表示する設定にすると良いでしょう。

Eclipse, 隠しファイルの設定

Package Explorer上のパッケージ表現

Package Explorer上のパッケージ表現をフラットにするか？階層構造にするか？選択できます。

デフォルトではフラットになっていますが、階層構造の方が扱いやすいと思う方は、階層構造に設定すると良いでしょう。

Eclipse, Package Explorer上のパッケージ表現
