パッケージ見た目表現とは？

Eclipse の Package Explorer上では、パッケージの表示形式 (Package Presentation) が複数あります。

フラット (デフォルト)

階層構造

ワークスペース設定

デフォルトから変更する設定はありますが、ワークスペース設定なので新たにワークスペースを作るたびに設定し直す必要があります。

階層構造に変更する

階層構造に変更する場合は、Package Explorerの右上の「白下三角」のマークをクリックしてメニューを出して... (Eclipseのバージョンによっては、マークが縦の点点点になっているかもです)

Package Presentation の Hierarchical を選択します。

すると、パッケージ見た目表現が、階層構造になります。
